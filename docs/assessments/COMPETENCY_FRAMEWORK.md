## Assessment Methodology

### Question Types

Each competency is assessed through **4 question types**:

1. **Multiple Choice** - Knowledge and understanding
2. **Likert Scale** - Self-assessment of frequency/intensity
3. **Scenario-Based** - Application in real situations
4. **Self-Reflection** - Personal evaluation

### Scoring System

- **Per Question**: 0-4 points
- **Per Competency**: 0-16 points (4 questions × 4 points)
- **Per Faculty**: 0-192 points (12 competencies × 16 points)
- **Overall Score**: 0-1,536 points (8 faculties × 192 points)

### Score Interpretation

| Score Range | Rating | Description |
|-------------|--------|-------------|
| 90-100 | Excellent | Mastery level competence |
| 80-89 | Good | Strong competence with minor gaps |
| 70-79 | Average | Adequate competence, room for growth |
| 60-69 | Fair | Basic competence, significant development needed |
| 0-59 | Needs Improvement | Substantial development required |
